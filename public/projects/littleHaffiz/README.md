### Rebuilding from Flutter source
This folder contains the release artifacts only. To modify logic/UI:
1. Open the original Flutter project (not included here).
2. Update the code as needed.
3. Run `flutter build web --release --base-href /projects/littleHaffiz/`.
4. Replace the generated `build/web` output with the contents of this directory.
